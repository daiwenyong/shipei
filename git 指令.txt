步骤：
初始化：
  git init  
		
添加内容:有工作区转暂存区
  git add . 

推送：由暂存区转分支区		
git commit -m '备注功能'

删除指定的远程：
   git remote rm origin	

添加远程仓库：
   git remote add origin <你的项目github地址>

上传代码到github之前需要先pull  下拉远程
   git pull origin master	

如何解决failed to push some refs to git：出现错误的主要原因是github中的README.md文件不在本地代码目录中
   git pull --rebase origin master	

推送到github：
git push -u origin master



细节：
	作为 git 的基础配置，作用是告诉 git 你是谁，你输入的信息将出现在你创建的提交中。
	git config --global user.name "你的名字或昵称"
	git config --global user.email "你的邮箱"
