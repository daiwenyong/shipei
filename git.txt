步骤：
初始化：
  git init  
		
添加内容:有工作区转暂存区
  git add . 

推送：由暂存区转分支区		
git commit -m '备注功能'

删除指定的远程：
   git remote rm origin	

添加远程仓库：
   git remote add origin <你的项目地址>

推送到github：
git push origin master



细节：
	作为 git 的基础配置，作用是告诉 git 你是谁，你输入的信息将出现在你创建的提交中。
	git config --global user.name "你的名字或昵称"
	git config --global user.email "你的邮箱"
